# PleaseGiveMeAssets Frontend Project
- **공통 사항**
  - 배포전 프로젝트 자동정렬 및 import 필수!!!(VS Code)
    - 정렬
      - macOS : Option + Shift + F
      - Window : Shift + Alt + F
  - Git 배포시 commit message는 필수로 입력한다.
  - 문의 사항은 Git issue나 comment, Slack을 통해 질문한다.
  - README.me 파일은 생각나는 기타 사항들을 모두 작성해주면 됩니다.
